# AideCMS 官方网站

基于 Vite + Vue 3 + TypeScript 构建的 AideCMS 项目官方宣传网站。

## ✨ 特性

- 🚀 Vite - 极速的开发体验
- ⚡️ Vue 3 + TypeScript - 现代化的前端技术栈  
- 🎨 响应式设计 - 完美适配各种设备
- 🌈 现代 UI - 渐变色、动画、毛玻璃效果
- 📱 移动端优化 - 触控友好的交互体验

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

访问 http://localhost:5173
